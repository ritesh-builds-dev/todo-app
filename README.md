## 📋 Project Overview
The Todo App is a practical front-end solution designed to help users stay organized. The project emphasizes core JavaScript functionality, ensuring that tasks are managed efficiently with a minimalist interface that works perfectly across all devices.

## ✨ Key Features
- **Data Persistence:** Integrated with **LocalStorage** to ensure your tasks remain saved even after closing the browser or refreshing the page.
- **Dynamic Filtering:** Real-time task filtering allowing users to toggle between **All**, **Active**, and **Completed** views.
- **Smart Task Counter:** An automated counter that tracks pending tasks to keep you on top of your goals.
- **Pixel-Perfect Responsiveness:** Meticulously optimized for all viewports, ensuring a premium feel from large desktops down to **320px** mobile screens.
- **Clean UI Logic:** Features custom validation to prevent empty tasks and a "Clear Completed" function for bulk management.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure for optimal accessibility and clean DOM organization.
- **CSS3:** Modern layout techniques using **Flexbox** and custom media queries for a responsive design.
- **Vanilla JavaScript (ES6+):** Pure logic for DOM manipulation, event handling, and state management without external frameworks.

## 📸 Design Highlights
- **Minimalist Aesthetic:** A distraction-free, monochrome design tailored for productivity.
- **Performance First:** Lightweight code execution with zero dependencies for instant loading times.
